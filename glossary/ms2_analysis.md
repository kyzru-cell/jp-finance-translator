# 任务2：核查现有词库的错误

## 一、订单类型（注文方法）

| 词库条目 | 判定 | 说明 |
|---------|------|------|
| 订单类型→注文方法 | 💡建议改进 | PDF中"注文方法"确实作为大标题使用，但具体选择订单种类时PDF用的是「注文の種類」。建议：大分类用「注文方法」，具体下拉选项说明用「注文の種類」或「注文種類」。词库注释中写"×注文タイプ、注文種類"过于绝对，PDF实际多次使用「注文の種類」 |
| 市价单→成行 | ✓正确 | PDF确认「成行」 |
| 限价单→指値 | ✓正确 | PDF确认「指値」 |
| 止损限价单→逆指値(指値) | ✓正确 | PDF中逆指值可设定为指值或成行执行 |
| 止损市价单→逆指値(成行) | ✓正确 | 与PDF描述一致 |
| 竞价市价单→寄り成行 | ✓正确 | PDF未直接出现但为日本市场标准用语 |
| 竞价限价单→寄り指値 | ✓正确 | 同上 |
