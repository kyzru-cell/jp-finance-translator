## 任务2：与已有词库对比，发现表达差异

| 概念 | 楽天表达 | SBI表达 | 建议采用 |
|------|----------|---------|----------|
| 平仓卖出（信用买建） | 転売 | 返済売 | 楽天「転売」更简洁，但SBI「返済売」更直观易懂，建议视语境选用，词库两者并收 |
| 平仓买入（信用卖建） | 買戻 | 返済買 | 同上，楽天「買戻」更简洁，SBI「返済買」更明确方向 |
| 平仓买入（先物卖建） | 買戻 | 決済買 | SBI「決済買」用于先物OP，与信用的「返済買」区分，建议按品种分别采用 |
| 平仓卖出（先物买建） | 転売 | 決済売 | SBI「決済売」用于先物OP场景，建议按品种分别采用 |
| 自选股列表 | お気に入り | 登録銘柄 | SBI「登録銘柄」更为业界通用，建议采用 |
| 预设订单/条件待发 | 予約注文 | 準備注文 | SBI独有概念名称，楽天对应功能为予約注文，建议词库两者并收并注明券商差异 |
| 篮子订单（批量预设） | — | バスケット注文 | SBI先物OP特有，楽天无完全对应功能名称 |
| 追加保证金（信用） | 追証 | 追加保証金 | 楽天「追証」为简称，SBI用全称「追加保証金」，建议统一为「追証（追加保証金）」 |
| 追加保证金（先物OP） | 追証 | 追加証拠金 | SBI先物OP明确区分用「証拠金」，建议按品种区分收录 |
| 全部挂单/全档行情 | — | 全板サービス | SBI付费服务名称，楽天无完全对应功能名 |
| 快速交易 | エクスプレス注文 | スピード注文 | 概念相同，SBI「スピード注文」更直观通用，建议采用SBI表达 |
| 行情提醒 | アラート | アラート | 两社一致，无差异 |
| PTS夜间交易 | PTS | PTS | 两社一致，无差异 |
